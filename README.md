# Stock_Predictor
A Super Powered Stock Predicting System, built using Extreme Learning Machine (ELM) and Brain Emotion Learning (BEL) for the stock prediction. Also used the Technical Analysis like MACD, Price Action Trading, Resistance and Support, Market Momentum analysis. The product can easily fetch the stock details using Alpha Vantage Library

---

## 📌 Overview

This project leverages **Extreme Learning Machine (ELM)** and **Brain Emotional Learning (BEL)** algorithms to predict stock prices by learning both numerical patterns and market behavior.

Along with ML models, the system integrates key **technical indicators** such as:

* 📈 MACD (Moving Average Convergence Divergence)
* 📊 Support & Resistance Levels
* 📉 Price Action Trading
* ⚡ Market Momentum Analysis

Real-time stock data is fetched using the **Alpha Vantage API**, enabling dynamic and up-to-date predictions.

## 🏗️ System Architecture

1. Fetch real-time stock data from Alpha Vantage
2. Preprocess and clean the data
3. Apply technical indicators (MACD, Momentum, etc.)
4. Train models using ELM and BEL
5. Generate predictions
6. Display results through UI

---

## 📊 How It Works

* The system collects historical and real-time stock data.
* Technical indicators help identify trends and patterns.
* ELM provides fast learning with minimal computational cost.
* BEL enhances prediction by modeling emotional market behavior.
* The combined output improves forecasting reliability.

## 🚀 Getting Started

### Prerequisites

* Python installed
* API Key from Alpha Vantage

### Installation

```bash
git clone https://github.com/your-username/stock-prediction-system.git
cd stock-prediction-system
pip install -r requirements.txt
```

### Run the Project

```bash
python app.py
```
